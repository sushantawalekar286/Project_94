# Run Digital Waiter System in VS Code

Open the project folder in VS Code, then paste these commands into the terminal one by one:

```powershell
npm install
```

```powershell
npm run install-all
```

```powershell
npm run dev
```

If you want to run backend and frontend separately:

```powershell
cd backend
npm run dev
```

```powershell
cd frontend
npm run dev
```
